# 簡介:  

本系統致力於將LLM與運動情蒐流程結合，針對使用者輸入的問題自動生成情蒐報告，輔助提升情蒐效率與品質。  
  
現階段是針對2023年世界棒球經典賽美國隊的全部15位投手進行情蒐，使用者透過LINE傳送訊息至本系統後，  
系統會針對訊息中單一個別球員前一年(2022年)的賽季表現進行分析，生成詳盡的情蒐報告。
